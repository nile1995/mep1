# 9.1.1 锁定构件
<br/>

&emsp;&emsp;锁定专业构件界面如图9.1.1\-1所示，在编辑机电专业模型时，可通过锁定所选专业构件避免修改其他专业的构件。


:-: ![](images/509.png)
图9.1.1\-1锁定专业构件界面

&emsp;&emsp;\[建筑专业构件\] \[结构专业构件\] \[机电专业构件\] \[装配式设计构件\] \[通用构件\]用于选择需要锁定的专业构件，可多选也可单选。

&emsp;&emsp;\[确定\] 即在当前模型中，按所选专业对构件进行锁定。

&emsp;&emsp;\[取消\] 即在当前模型中，按上一次所选专业对构件进行锁定。
