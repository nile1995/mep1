
# 捕捉设置
<br/>
&emsp;&emsp;捕捉开关在命令提示行下方，如图 2.3-1。点击黑色倒三角，可以快速勾选对象捕捉点的类型。也可点击最下角的“对象捕捉设置”进行精细设置，如格栅捕捉设置、极轴追踪设置、捕捉类型点设置、靶区设置和专业捕捉设置。

&emsp;&emsp;单击正常显示为关，单击灰度显示为开。
