#  7.1.4 线管空间连接
<br/>

&emsp;&emsp;使用线管空间连接命令，可以轻易完成空间交叉线管之间的连接。操作步骤如下：点取需要连接的交叉管道，软件会自动用三通和竖管将空间管道连接起来，打断的线管和竖管之间用弯头或者接线盒连接，具体用哪种构件可以通过工程设置进行设定。如图表7.1.4\-1，7.1.4\-2和图表7.1.4\-3所示。
